Welcome

This is a little service for downloading data from Twitter and Reddit. The service is built and provided for Navigating Complexity (aka. NavCom) students by the ETHOS Lab.

After selecting either Twitter or Reddit as your data source, defining your query and clicking the Submit button, tabular data will be downloaded in a CSV file for further processing.

To guide use in your course projects, and in case of technical issues, please speak with the NavCom Teaching Assistants.

Please consider the software to be of experimental quality. The software is currently provided AS IS, without warranties. Currently both the Twitter and Reddit interfaces have extra limits in place for development purposes, and do not return a lot of data. The source code of this software is available online.

Thank you to Open Source libraries getOldTweets3 and PRAW, as well as the Python community at large.
